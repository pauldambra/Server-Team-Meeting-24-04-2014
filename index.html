What frameworks are there

QUnit

  • I've never used it.
  • They use it to test EmberJs and JQuery.
  • depends on running in a browser (ish)
  • Supported by Resharper.
							
test("can compare things", function() {
  ok(1 == "1", "Passed!");
});

Mocha

  • Similar to Jasmine but doesn't come with a mocking or assertion framework built in
  • It handles asynchrony really well
  • Not supported by R#
							
var expect = require("chai").expect;
describe('can compare things', function(){
    it('should be true that 1 == "1"', function(){
      expect(1).to.equal("1");
    });
});

Jasmine

  • Similar to Mocha but comes with a spy (like a stub or mock) and assertion framework
  • Jasmine 2.0 handles asynchrony (apparently) as well as Mocha
  • Supported by Resharper!!!
							
describe('can compare things', function(){
    it('should be true that 1 == "1"', function(){
      expect(1).toEqual("1");
    });
});

Why Jasmine?

  • Did I mention it's supported by R#
  • and comes with a HTML runner so it can easily be hooked into a Jenkins build using PhantomJS and Chutszpah

Definitely go and play with other test frameworks and show me I'm wrong!

Resharper Gotcha!

R# has to be able to discover the JS files used in a test and uses reference path comments to do that

				
///
///

describe("Shopping Cart", function () {

  beforeEach(function() {
  });

  it('should have some tests');
});

NB ignore the trailing closing reference tags! That's the highlighting library getting upset by VS specific things

What are Katas and Why do them?

Shopping Cart Kata

Implement the code for a supermarket checkout that calculates the total price of a number of items.

Goods are priced individually, however there are weekly special offers for when multiple items are bought. For example “Apples are 50 each or 3 for 130″.

Weekly offers change frequently.

The checkout accepts the items in any order, so that if we scan a pack of Biscuits, an apple and another pack of biscuits, we’ll recognise two packs of biscuits and apply the discount of 2 for 45.

SKU Unit Price Special Offer
A99 50 3 for 130
B15 30 2 for 45
C40 60
T34 99

What tests should there be?

				
describe("Shopping Cart", function () {

  beforeEach(function() {
  });

  it('should have some tests');
});
